# password generator 随机密码生成器

Easy generator with simple UI

RANDOM PASSWORD

based on python random, string, tkinter



## Use

### commander version

use python run

```bash
git clone 

cd ~\password

python .\commander\generator-cmd.py
```

### main version

use python run

```bash
git clone

cd ~\password

python .\ui.py
```
